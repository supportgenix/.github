## About Us

SupportGenix builds customer support solutions that help businesses respond faster, reduce support costs, and turn customers into loyal advocates. Whether you run a WordPress site, manage a documentation portal, or need a full SaaS support platform, we have a tool for you.

## Our Products

### 🎫 SupportGenix for WordPress

A full-featured WordPress support ticket and knowledge base plugin with built-in ticketing, email piping, AI support, and integrations with WooCommerce, Easy Digital Downloads, Slack, WhatsApp, and more.

[Learn More →](https://supportgenix.com)

### 📚 SupportGenix Docs (Free & Open Source)

A free, production-ready Astro theme for documentation sites, help centers, and knowledge bases. Static, fast, and fully customizable with MDX, Pagefind search, and Tailwind CSS.

[View on GitHub →](https://github.com/supportgenix/supportgenix-docs) · [Documentation →](https://github.com/supportgenix/supportgenix-docs#readme)
